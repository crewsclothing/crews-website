### Setup Instructions

1. Open 2 terminals:
   - Terminal 1:
     cd server
     npm install
     npm start

   - Terminal 2:
     cd client
     npm install
     npm start

2. Make sure MongoDB is running (mongod)
3. Check .env file in server folder for config (DB URI, Email settings, etc)
